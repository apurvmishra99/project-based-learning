## Created A Twitter Bot

Can access twitter account and read tweets from feed to perfrom :

    1.Follow
  
    2.Retweet
  
    3.Favourite
  
    4.Respond
  
    5.Search

Used : Tweepy library and Tkinter to make a GUI for the user.
